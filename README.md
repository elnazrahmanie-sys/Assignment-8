# Accessories Shop

## Project Description

Accessories Shop is a multi-page responsive online accessories store created using HTML, CSS, Bootstrap, and JavaScript.

This project includes a Home page, About page, Contact page, and Shopping Cart page. Users can browse products such as rings, bracelets, watches, handbags, and hair accessories.

---

## Features Included

- Responsive multi-page website  
- Bootstrap navigation bar  
- Product cards using Bootstrap cards  
- Add to Cart functionality  
- Shopping cart page with quantity controls (+ / -)  
- Total price calculation using JavaScript  
- Contact page with interactive style suggestion form  
- About page with company story and team section  
- Footer and smooth page navigation  
- Hover effects and transitions  
- CSS animation  
- Mobile-friendly responsive design
